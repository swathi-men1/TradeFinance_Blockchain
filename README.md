**Trade Finance Blockchain Project**

**Project Description**

This project is a Trade Finance Management System developed using Django.
The main aim of this project is to manage trade activities securely using a role-based system.
Different users such as Bank, Corporate, and Auditor can access the system and perform their specific tasks like managing trades, monitoring risks, and maintaining ledger records.
The project follows a blockchain-based workflow idea to improve transparency and tracking of transactions.

**Features**
-Secure login and authentication
-Role-based access control
-Trade Card dashboard
-Ledger management module
-Risk monitoring module
-User-friendly interface

**User Roles**
Bank – Manages and reviews trade transactions
Corporate – Creates and tracks trade details
Auditor – Monitors risk and ledger activities

**Technologies Used**
-Python
-Django Framework
-HTML & CSS
-PostgreSQL
